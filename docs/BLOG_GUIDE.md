# 博客维护指南

复制 `docs/templates/blog.mdx` 到 `src/content/blog/`。文件名会成为 URL；发布前检查标题、摘要、分类、标签和日期。

- `draft: true` 为草稿，不生成生产页面。
- `featured: true` 在博客推荐区展示。
- 封面放在 `public/images/blog`；省略时使用站点默认分享图。
- 代码使用带语言标识的围栏，如 ` ```ts `。
- 表格、脚注和数学公式已配置；块级公式用 `$$...$$`。
- 提示块可在 MDX 中导入 `Callout.astro`，参考示例文章。
- `canonical` 仅用于内容原先发布在其他地址时；通常留空。

分类应保持少而稳定，标签可更具体。图片必须写替代文本，外部资料需核实引用与授权。
