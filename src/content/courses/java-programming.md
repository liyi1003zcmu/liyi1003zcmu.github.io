## 课程简介

围绕可维护软件展开 Java 核心语法、面向对象思想和基础工程实践。

## 适用对象

已有基础编程经验，希望学习静态类型面向对象语言的学习者。

## 课程目标

- 建立对象建模意识；
- 正确使用集合、异常和文件 API；
- 编写结构清晰且可测试的程序。

## 先修要求

理解变量、流程控制、函数与数组。

## 教学安排

讲授、代码走查与结对练习相结合，共 16 个教学周。

## 章节目录

1. Java 平台与工具链
2. 类、对象与封装
3. 继承、接口与多态
4. 泛型与集合
5. 异常、文件与测试
