课程介绍

  • 学什么？---- 程序设计，Java
  • 怎么学？---- 多看，多想，多敲
  • 学什么？---- 学会”学习“和”思考“
  • 怎么学？---- 投入时间和精力，投入自己的手和脑

课程目标

  • 认识Java，又多学一种计算机方言
  • 理解Java，更深入地理解面向对象，理解如何深入地分析问题
  • 理解程序和算法，尝试系统性思维，独立构建应用系统解决综合性问题

关于教材

  • Java程序设计, 杨文艳，田春尧. 北京理工大学出版社, 2018

关于参考教材

  • Introduction to Java Programming and Data Structures, Comprehensive Version, 12th ed. Pearson, 2020, Y. Daniel Liang, Pearson, 2020
  • Learning Java: An Introduction to Real-World Programming with Java 5th Edition, Marc Loy, Partrick Niemeyer, Daniel Leuck, O'Reilly, 2020
  • Thinking in Java, 4th ed., Bruce Eckel, Pearson, 2006
  • Core Java Volumn (I && II), 12th ed. , Cay S. Horstmann, Person, 2021

关于在线资源

  • 国家级精品课程--零基础学Java
  • 面向对象程序设计——Java语言

课程前置要求

  1. 学过程序设计基础，会写“好”的代码
  2. 了解常见数据结构和算法
  3. 了解面向对象的基本概念，熟悉C++面向对象语法特征及实现
  4. 用C, C++或其它语言写过小型的完整应用程序

本课程要求

  • 掌握基础概念和理论，学会正确的程序编写、调试、运行方法
  • 能够正确使用Java常见类库编写Java程序
  • 独立完成课程实验，提升自己的代码量
