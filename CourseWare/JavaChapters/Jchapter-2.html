写一段程序，求给定半径的圆的面积

                    
/**
 CircleArea: Compute the area of a circle with given radius
*/

public class CircleArea{
  public static void main( String[] args ){
  // define the PI constant
  final double PI = 3.14159;

  // declare the radius variable
  double radius;

  // declare the area variable
  double area;

  // assign value to variable radius
  radius = 1.0;

  // assign value to variable area
  area = PI * radius * radius;

  System.out.println( "The area of circle with radius " 
                    + radius + " is " + area );
 }
}

标识符

标识符
Java中对各变量、方法、类、包、参数等命名时所使用的字符序列

  • 命名规范
    • 由字母、数字、下划线或美元符号组成
    • 不能以数字开头
    • 区分大小写
    • 长度无限制
    • 不能使用Java中的保留关键字
    • 为便于系统后期测试和维护，建议采用有意义的单词或词组

变量-命名规则

  • 变量名由任意多个字母、数字、下划线(_)、美元符号($)组成
  • 变量名应由字母、下划线、美元符号开头(注意什么不能开头？)
  • 变量名严格区分大小写
  • Java中的关键字或保留字不应用作变量命名
  • 变量名通常以小写字母开头
  • 类名首字母为大写(注意这里对变量名和类名的描述)
  • 美元符号用于语言内部处理，如机器自动生成的代码

变量--使用

  • 变量是程序中最基本的存储单元，其要素包含变量类型，变量名和变量作用域
  • 每个变量都属于特定的数据类型，使用前需先声明，必须赋值后才能使用
    • 
double radius; radius = 10.0;
                            
    
double radius = 10.0;
  • 变量使用需注意变量的作用域，即定义变量的有效范围 VariableScope.java

常量

  • 程序运行过程中不变的量，值不能改变
  • 用final关键字标识
  • 一般常量定义用大写字母命名，单词与单词之间用下划线加以分割

public static final double PI = 3.14159;
                        

static final int DAY_OF_YEAR = 365;

数据类型--八种基本数据类型

  • 逻辑(boolean)
  • 整数(byte, short, int, long)
  • 浮点数(float, double)
  • 字符(char)

数据类型--逻辑

boolean

  • 数据值为true和false
  • 不允许赋值为TRUE和FALSE
  • 也不允许以0及非0的整数替代true和false

数据类型--数值类型取值

数据类型--整型

byte, short, int, long

  • 各整型有固定的表数范围和字段长度，保证程序的可移植性
  • 整型常量有三种表示形式
    • 十进制，如12, -314, 0
    • 八进制，要求以0开头，如012
    • 十六进制，要求以0x或者0X开头，如0x12
  • 整型常量默认类型为int，若声明为long类型，需在后面加'l'或'L'
    • 
int i1 = 600;
long l1 = 888888888888L;

数据类型--浮点

float, double

  • 各浮点类型有固定的表数范围和字段长度，保证程序的可移植性
  • 浮点数常用表示方法
    • 十进制，如3.14 314.0 .314
    • 科学计数法，如3.14e2 3.14E2 3.14E-3

数据类型--字符型

char

  • 用于表示单个字符
  • 字符型常量为使用单引号括起来的单个字符，如
    • 
char grade = 'A'; char cChar = '中';
  • Java中字符采用Unicode编码，用两个字节(16个二进制位)表示单个字符
    • 
char c1 = '\u4E2D';
  • 转义字符'\'的作用是将其后的字符转变为其它含义，如
    • 
char c2 = '\n'; char c3 = '\\';
